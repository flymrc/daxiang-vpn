## 使用方法

```powershell
.\zhvpn.exe login <上面任意一个令牌>
.\zhvpn.exe start
.\zhvpn.exe status
```

启动后把浏览器或系统代理设为 `127.0.0.1:7890`，即可通过日本住宅 IP 上网。
详细步骤见《纵横 VPN 客户端使用指南》。
